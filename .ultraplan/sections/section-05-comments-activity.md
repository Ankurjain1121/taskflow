# Section 05: Comments & Activity Log

## Overview
Add the ability to comment on tasks with @mentions that notify the mentioned person. Build a per-task activity timeline showing who did what and when (created, assigned, moved, commented, file attached). Backend uses Rust (Axum + SQLx) with REST JSON endpoints. Frontend uses Angular 19 standalone components with Tailwind CSS v4 and Angular Material.

## Risk: [green] - Standard features with clear patterns

## Dependencies
- Depends on: 03, 04
- Blocks: 08, 11
- Parallel batch: 3

## TDD Test Stubs
- Test: User can add a comment to a task
- Test: @mentioning a user in a comment triggers a "mention-in-comment" notification for that user
- Test: @mentioning a user in a comment triggers a "task-commented" notification for task assignees
- Test: Activity log records task creation with correct timestamp and user
- Test: Activity log records task movement between columns
- Test: Activity log records assignment changes
- Test: Comments are ordered chronologically with newest at bottom
- Test: Comment creation triggers WebSocket broadcast on board:{boardId} channel for real-time sync
